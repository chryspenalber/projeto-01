História da Web

Nessa aula, aprendemos sobre um pouco sobre a história da Web. Iniciamos com os Primeiros Computadores onde foram mencionadas as suas gerações.

1ª geração:

  • Mark I (1944)
  • Colossus (1946)
  • ENIAC (1946)

2ª geração:

Entre 1959 a 1965

Mudança de válvula para transistores, que diminuiu os Computadores.

3ª geração:

Entre 1965 a 1970

Criação dos Circuitos Integrados (Chips) - que diminuiram ainda mais os Computadores.

Um CHIP é composto por transistores, capacitores e resistores.

4ª geração:

Entre 1971 a dias atuais

Marcado pelo surgimento do PC e a Apple cria o Mouse.

5ª geração:

Futuro

Acredita-se que já estamos vivendo a 5ª geração. Estão inseridas nela a AI e computadores quânticos, etc...


Seguimos para falar sobre o surgimento da Internet.

  • Surgiu na década de 1950 - Guerra Fria entre EUA e URSS.
  • Criado o ENIAC.
  • A DARDA surge.
  • Surge a ARPANET - Sistema de comunicação em pacotes, enviados um a um e que permitia múltiplos destinos, o "Avô dos roteadores"
  • Acontece a primeira comunicação em 29/10/1969. Na tentativa de enviar "LOGIN" somente foi recebido LOGIN.
  • Vinton Cerf e Robert Kahn criam o TCP/IP (protocolos de comunicação).
  • Tim Bernes-Lee une o hipertexto com o TCP/IP e cria a WWW.

Por último falamos sobre Grandes Nomes.

  • Grace Hopper
  • Joseph Licklider
  • Robert Kahn
  • Tim Bernes-Lee
  • Marc Andreessen
  • Bill Gates
  • Steve Jobs

O que são Servers?

Os servidores podem ser internet ou local (computador mais robusto).

Data Center é onde tem vários servidores.

Tipos de servidores
  • Servidor Proxy - acessa internamente certas informações antes de acessar a internet.
  • Firewall - barreira de segurança para acesso externo.
  • Web Server - sites
  • E-mail Server - armazenamento de e-mails.
  • Database Server - banco de dados do servidor.
  • DNS - responsáveis por fazer com que o seu domínio seja localizado na internet.
Hospedagem de sites

Colocar os arquivos do site num servidor para ser disponibilizado na internet para o cliente.

Conexão FTP

Conexão feita com o servidor para envio de arquivos.

Linguagens de Programação

Linguagem escrita e formal que especifica um conjunto de instruções e regras para gerar um programa.

Linguagens mais populares
  1. Javascript
  2. Python
  3. Java
  4. PHP
  5. C#
Server Side

Linguagens para rodar no servidor. Exemplo: PHP, Ruby, Java, Javascript.

Client Side

O Javascript executa dos dois lados e é o mais utilizado no client side.

V8 é o mecanismo que interpreta e executa o Javascript no navegador.
